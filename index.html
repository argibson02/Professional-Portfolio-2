MY PROJECTS

Reimagining virtual museums

The 3D Art Gallery

The 3D Art Gallery aims to improve upon the virtual museum experience by displaying museum paintings in a 3D space while exaggerating the texturing on the picture to give a feeling that you’re right in close. Users can grab, manipulate, zoom in, zoom out, and see how light interacts with a piece of art.

Built with: Javascript, jQuery, three.js, HTML, CSS, Materialize, Rijksmuseum API, MET API, and QR Code Generator API.

Weather at a glance

Weather Dashboard

This is a simple weather dashboard web app that utilizes OpenWeather API to show the current weather and a five-day forecast for any city. You can also search previously entered locations with ease by pressing on the recently searched buttons.

Built with: Javascript, jQuery, HTML, CSS, OpenWeather One Call API, OpenWeather Geolocation API, and Moment.js.
